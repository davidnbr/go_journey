How It Works

01

Read the exercise README

Each exercise has objectives, concept explanations, and hints behind spoiler gates.

02

Fill in the stub

Every exercise.go has TODO markers. Your job is to make the tests pass.

03

Run the tests

Use make check PHASE=… EX=… to get instant feedback on each exercise.

04

Review & repeat

Each phase ends with a spaced-repetition review exercise mixing everything covered so far.

Quick Start

bash — requires Go 1.22+
# Clone and enter the repo
$ git clone https://github.com/davidnbr/go_journey.git && cd go_journey

# Run a single exercise
$ make check PHASE=phase1-foundations EX=01-hello-world

# Run all tests for a phase
$ make test-phase PHASE=phase1-foundations

# Check overall progress
$ make progress
